# 🍽️ Reestar – Simple Food Order Website

Reestar is a **simple food ordering website UI** built using **HTML and CSS**.
This project was created as a practice project while learning **basic web development and layout design**.

The website displays food items with images, descriptions, and an order button along with a contact form.

---

## 🚀 Features

* Clean food card layout
* Navigation header (Home | Recipes | Contact)
* Food item display with images
* Order buttons for each food item
* Simple contact form
* Built using only **HTML & CSS**

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
